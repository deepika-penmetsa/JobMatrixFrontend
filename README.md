## ðŸ›  Getting Started  

### Prerequisites  
- **Node.js** (v16+)  
- **npm** (v8+)  

### Installation  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/RANJITH-REDDY-N/JobMatrixFrontend  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd JobMatrixFrontend  
   ```  
3. Install dependencies:  
   ```bash  
   npm install  
   ```
